## 2. 파일 입출력 응용
- 여러 과목 점수 기록
```python
aos = open("scores.txt", "a", encoding="utf8")
while True:
aa = input("과목명 입력 (종료: p): ")
if aa == "p":
break
ab = int(input("점수 입력: "))
print(f"{aa} : {ab}", file=aos)
aos.close()
```
- 읽어서 점수만 추출 후 총점/평균 계산
```python
sos = open("scores.txt", "r", encoding="utf8")
scores = []
while True:
line = sos.readline()
if not line:
break
score = int(line.split(":")[1].strip())
scores.append(score)
sos.close()


total = sum(scores)
average = total / len(scores)
```


## 3. split() vs strip()
- `split(":")` → 문자열 내 `:` 기준으로 분리, 리스트 반환
- `strip()` → 앞뒤 공백 제거
- 점수만 뽑을 때: `int(line.split(":")[1].strip())`


## 4. 조건문 활용 예시
```python
if average >= 90:
grade = "A"
elif average >= 70:
grade = "B"
elif average >= 50:
grade = "C"
else:
grade = "D"
```
